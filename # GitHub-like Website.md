# GitHub-like Website

A simplified GitHub clone using multiple technologies.

## Features
- User authentication with cookies
- Basic repository management
- SSH integration for Git operations
- Web interface for browsing repositories

## Requirements
- Go 1.16+
- Python 3.6+
- Bash
- SSH server
